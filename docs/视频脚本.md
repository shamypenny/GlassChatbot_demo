## 录制指南

### 录制准备

1. **环境准备**
   - 关闭无关软件和通知
   - 清理桌面和浏览器书签
   - 准备好演示数据

2. **软件设置**
   - 浏览器全屏模式
   - 隐藏书签栏
   - 字体大小适中

3. **录制工具**
   - Windows: OBS Studio / 录屏大师
   - Mac: QuickTime / ScreenFlow
   - 推荐分辨率: 1920x1080

### 录制要点

1. **操作流畅**
   - 提前练习操作流程
   - 操作速度适中
   - 避免多余操作

2. **画面清晰**
   - 重点内容放大展示
   - 鼠标移动平滑
   - 点击有明显反馈

3. **时间控制**
   - 每个功能演示约30-45秒
   - 总时长控制在3-4分钟

### 后期处理

1. **剪辑**
   - 删除多余操作
   - 添加转场效果
   - 调整节奏

2. **配音**
   - 使用专业录音设备
   - 语速适中，清晰有力
   - 可使用AI配音工具

3. **字幕**
   - 添加关键信息字幕
   - 字体清晰易读
   - 位置不遮挡内容

4. **背景音乐**
   - 选择轻快专业的BGM
   - 音量不盖过旁白
   - 风格与内容匹配
